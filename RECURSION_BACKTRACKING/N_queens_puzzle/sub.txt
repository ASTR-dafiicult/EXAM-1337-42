Write a function that returns an array of integers containing all values between start and end (exclusive of end).

The function must be declared as follows:

int	*ft_interval(int start, int end);
📌 Requirements

The array must contain all integers from start up to end - 1.

If start is greater than or equal to end, the function must return NULL.

Memory must be allocated using malloc.

The size of the array must be exactly end - start.

💡 Examples
Function Call	Expected Result
ft_interval(3, 7)	{3, 4, 5, 6}
ft_interval(-2, 2)	{-2, -1, 0, 1}
ft_interval(5, 5)	NULL
ft_interval(7, 3)	NULL
